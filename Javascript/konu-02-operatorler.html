Javascript - Operatörler

Aritmetik Operatörler

+, -, *, /, %, ++, -- sembolleri ile hesaplama işlemlerini gerçekleştirebiliriz.

Atama Operatörleri

=, +=, -=, /=, %=

Karşılaştırma Operatörleri

== : eşit mi, === : değerleri veri tipleriyle birlikte karşılaştırır, != : eşit değilse, !== : değerler ve tipler aynı değilse, > büyüktür, < küçüktür, >= büyükse veya eşitse, <= küçük veya eşitse, ? ternary operatörü

Mantıksal Operatörler

&& : ve, || : veya, ! : değilse
